# BMW Used Car Market Analysis

## About
A beginner Python data analysis project exploring a BMW used car dataset using Pandas and Matplotlib.

## Features
- Count unique BMW models
- Find the most common model
- Calculate average prices
- Compare fuel types
- Visualize the top 5 most expensive models

## Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
BMW used car listings.

## How to run

1. Clone the repository
2. Install the required libraries:
pip install pandas matplotlib

3. Open 'notebook.ipynb' in Jupyter Notebook or VS Code.
